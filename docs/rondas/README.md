# Rondas: trackers de trabajo VIVOS (Landing)

Cada ronda/auditoria de trabajo deja aqui su tracker: orden de trabajo, hallazgos y estado punto por punto (`[ ]` / `[wip]` / `[hecho]`).

- Nombre: `YYYY-MM-DD_TEMA.md`.
- El tracker registra tambien las decisiones que toma el dueno (se copian a `../PENDIENTES_DUENO.md`) y los gotchas descubiertos (se copian a `CLAUDE.md` si son reglas permanentes).
- Al cerrar la ronda (todo `[hecho]` o los restos movidos a los `PENDIENTES_*`): archivar el tracker en `../archive/rondas/` (crear la carpeta cuando haga falta).
