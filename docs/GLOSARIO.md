# Glosario (Landing)

## Dominio

| Termino | Que es |
| --- | --- |
| Hub | La pagina raiz que agrupa los enlaces personales de Angel y lleva al portfolio y redes. |
| Legal (de app) | Par de documentos por app: Politica de Privacidad y Terminos y Condiciones, requeridos por las tiendas. |
| Directorio de enlaces | Lista de destinos (portfolio, GitHub, LinkedIn, contacto) que muestra el hub. |
| Validacion de tienda | Revision de Google Play / App Store que exige una URL publica y estable con los legales. |

## Tecnologias y servicios

| Termino | Que es aqui |
| --- | --- |
| Astro | Generador de sitios estaticos (SSG); mismo framework que el portfolio. |
| Tailwind 4 | Motor de estilos con tokens CSS-first (`@theme`), via `@tailwindcss/vite`. |
| GitHub Pages | Hosting estatico del sitio (repo publico). |
| Cloudflare | DNS del dominio, redireccion 301 `www` -> raiz, analitica sin cookies. |
| Podman | Contenedores para verificacion aislada (Node 24), sin Docker. |
